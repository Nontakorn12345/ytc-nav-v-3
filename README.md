# 🗺️ คู่มือการจัดการรูปภาพ YTC Nav (ฉบับเร่งด่วน)

เมื่อคุณได้รูปภาพมาแล้ว ให้ทำตามขั้นตอนดังนี้:

### ✅ 1. การเตรียมรูปแผนที่ (Main Map)
- **ชื่อไฟล์:** `campus-map.jpg`
- **ตำแหน่ง:** วางไว้ที่ `public/images/campus-map.jpg`
- **ขนาดแนะนำ:** 2000 x 1500 px เพื่อความคมชัดเวลาซูม

### ✅ 2. การเตรียมรูปอาคาร (Building Photos)
- **ชื่อไฟล์:** ต้องตรงกับ ID ในระบบ (b1, b2, b3, ...)
- **ตัวอย่าง:** 
  - ตึกช่างกลโรงงาน -> `b1.jpg`
  - ตึกเทคนิคพื้นฐาน -> `b2.jpg`
- **ตำแหน่ง:** วางไว้ที่ `public/images/`
- **ขนาดแนะนำ:** 1200 x 800 px (แนวนอน)

### ✅ 3. ตรวจสอบโครงสร้างโฟลเดอร์
โครงสร้างควรเป็นแบบนี้:
```text
project-root/
├── public/
│   └── images/
│       ├── campus-map.jpg
│       ├── b1.jpg
│       ├── b2.jpg
│       └── ... (จนถึง b15.jpg)
├── index.tsx
└── ...ไฟล์อื่นๆ
```

### 💡 คำแนะนำเพิ่มเติม
- หากไฟล์ภาพของพี่เป็นนามสกุลอื่น เช่น `.png` หรือ `.webp` พี่ต้องไปแก้ไขนามสกุลไฟล์ใน `constants.ts` ให้ตรงกันด้วยครับ
- ถ้าทำบนคอมพิวเตอร์ (Local) แล้วรูปไม่ขึ้น ให้ลอง **Refresh** หน้าเว็บ หรือกด `Ctrl + F5` เพื่อล้างแคชครับ
